Class Ext.ux.netbox.core.DynamicFilterModelView

Defined in DynamicFilterModelView.js

It implements the view for dynamic filters. Dynamic filters mean that the user is allowed to add, remove filter, seeing only the actual filters. It's possible to define more than one filter for the same field. It can be lazyly instantiated using dynamicFilter as xtype.

Example

The following code will instantiate a window with a DynamicFilterModelView inside using lazy initialization 
win=new Ext.Window({
  title: 'Filters',
  width:600,
  height:350,
  layout: 'border',
  closeAction: 'hide',
  items: [{ filterModel: filterModel,
    region: "center",
    xtype: 'dynamicFilter'
  }]
});
