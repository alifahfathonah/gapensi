Class Ext.ux.netbox.core.FilterHeaderPlugin

Defined in FilterHeaderPlugin.js

This is a plugin for an Ext.grid.GridPanel useful to highlight the columns of the grid on which there is at least a filter. It assumes that the id of the fields and the dataIndex of the columns are the same.

Example

grid = new Ext.grid.GridPanel({
  store: store,
  columns: [
    ....
  ],
  ...
  plugins: [new Ext.ux.netbox.core.FilterHeaderPlugin(filterModel)],
  ....
});
