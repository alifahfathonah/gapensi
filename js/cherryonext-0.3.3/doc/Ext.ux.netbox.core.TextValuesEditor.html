Class Ext.ux.netbox.core.TextValuesEditor

Extends Ext.ux.netbox.FilterEditor.
Defined in TextValuesEditor.js

This class extends Editor to manage an array of {label: originalValue, value: originalValue} as value. It should be used to manage all the form field that doesn't have a value different from the label (for example a TextField or a checkbox, but not a ComboBox where label and value are different)
