Class Ext.ux.netbox.string.StringField

Extends Ext.ux.netbox.core.Field.
Defined in StringField.js

This is the class that implements the field to use if the type is string. It contains as default the following operators:

  • STRING_EQUAL
  • STRING_DIFFERENT
  • STRING_CONTAINS
  • STRING_DOESNT_CONTAIN
  • STRING_STARTS_WITH
  • STRING_ENDS_WITH
The default operator is STRING_EQUAL. The STRING_LIST and STRING_NOT_LIST operations should be manually added if needed.
